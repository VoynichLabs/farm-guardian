# 05 — chick_distress

Chick distress peep (high-pitched repetitive descending peeps from a separated or cold chick). Cross-age-class alert: hens in the run may respond as a maternal-recall trigger.

Target: 3 exemplars, peak-normalized to -3 dBFS, clamped to 3-6 s. Public-domain or CC0.

Reference: distress peeps occupy 2-5 kHz with descending modulation, 100-250 ms per call (Marler & Evans / Royal Society review). Recordings dominated by ambient noise below 1 kHz are unsuitable.
