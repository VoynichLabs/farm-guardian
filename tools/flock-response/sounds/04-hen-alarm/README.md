# 04 — hen_alarm

Hen alarm call (the descending "buk-buk-buk-bagAWK" sequence on a startled hen). Conspecific, high-arousal. Expected: largest attention-toward-camera delta + alarm posture (plan H1).

Target: 3 exemplars, peak-normalized to -3 dBFS, clamped to 3-6 s. Public-domain or CC0. The alarm call has aerial vs ground variants — capture both if available; document which in the MANIFEST `notes` column.

**Welfare note:** combined with hawk-scream this is the highest-arousal category. The plan caps high-arousal trials at 1 per cohort per day; the experiment runner is responsible for enforcing that.
