# 08b — ambient

Farm ambient recorded with no birds present (or with the flock far from the speaker — shed wall between mic and birds). Played back through the same speaker as a real stimulus to test whether the birds respond to *the speaker activating at all* (single-source bias) versus the actual stimulus content.

Target: 3 exemplars, peak-normalized to -3 dBFS, clamped to 3-6 s.

These should be recorded *on the farm*, not sourced — the point is to match the ambient texture the birds already ignore. Record with the GWTC laptop's built-in mic (or a phone) at the speaker's eventual position.
